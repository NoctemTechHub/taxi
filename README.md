# Aydında Bu TAKSİ - Flutter Mobile App

Aydın şehrinde taksi hizmetlerini entegre eden modern mobil uygulama.

## 📱 Özellikler

- 🗺️ **Real-time Harita** - Google Maps entegrasyonu ile taksi konumları
- 👨‍💼 **Admin Paneli** - Taksi ve paket yönetimi
- 🚕 **Taksici Paneli** - Profil ve durum yönetimi
- 🔥 **Firebase** - Real-time database ve kimlik doğrulama
- ⚡ **Riverpod** - State management
- 🎨 **Material 3** - Modern UI design

## 🚀 Hızlı Başlangıç

1. **Kurulum Rehberleri**:
   - [FIREBASE_SETUP.md](FIREBASE_SETUP.md) - Firebase ve Maps kurulumu
   - [QUICKSTART.md](QUICKSTART.md) - Hızlı başlangıç
   - [IMPLEMENTATION_SUMMARY.md](IMPLEMENTATION_SUMMARY.md) - Implementasyon detayları
   - [example.md](example.md) - React Native örneği (Flutter'a uyarlama referansı)

2. **Dependencies Yükle**:
   ```bash
   flutter pub get
   ```

3. **Uygulamayı Çalıştır**:
   ```bash
   flutter run
   ```
